Binary Search

If you have a sorted list, one way to search for an element is by checking the middle element. If the middle element is larger than the desired element, you only search the first half. If not, you search the last half. The average run-time for binary search is O(log n).

Python

          def is_sorted(A):
            for i in range(len(A)-1):
              if A[i+1] < A[i]:
                return False
            return True

          # returns index of target
          def binary_search(A, target_value, n):
            if not is_sorted(A):
              print("Not sorted")
              return None
            L = 0
            R = n - 1
            while L <= R:
              mid = (L + R) // 2  # using floor division
              if A[m] < target_value:
                L = mid + 1
              elif A[m] > target_value:
                R = mid - 1
              else: # equality
                return mid

          test_list = [34, 2, 23, 12]
          test_list = sorted(test_list)
          print(binary_search(test_list, 34, 
              len(test_list)))
